- All tolerances, methods, configuration values, and outputs **must**

  institutional protocol, or your jurisdiction's regulatory regime.

## Patient privacy

Do **not** send Protected Health Information (PHI) — including
identifiable DICOM, log files, screenshots, or report PDFs — to
Radiant Medical Physics Consulting LLC, to any public distribution
channel (issue trackers, forums, email lists), or to any third-party
service via this software. Any test data shared for troubleshooting
must be fully de-identified first; the software includes a DICOM
anonymizer module for this purpose, but the user remains responsible
for verifying that anonymization is complete.

## Reporting issues

If you encounter a defect during evaluation, please contact
**tbernath@radiant-mpc.com** with a description of the behavior, the
software version, and (only if relevant and fully de-identified)
